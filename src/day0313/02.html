< HTML > 안에 포함되는 가장 기본적인 내용은 HEAD 와 BODY 엘리먼트(요소) 이다.

< HEAD > 제목을 포함하는 HTML 문서의 머리(HEAD) 부분임을 알려준다.
모든 HTML 문서에는 HEAD 가 필요하다.
일반적으로 HTML 에서는 대소문자를 구별하지 않는다. < HEAD > 와 < head > 는 같다

Document 02 -> 문서의 제목(TITLE) 부분을 나태내므로 전체적인 문서의 내용을 대표 할 수 있는 주제어를 담게한다.
문서를 표시하는 영역에는 나타나지 않으나,
웹 브라우저에 대체적으로 맨 위에 표시된다.
문서를 대표하는 목록, 북마크, 창 등으로 표시되고 검색에서도 사용되기 때문에 생략하지 않는것이 좋다.

< BODY > HTML 문서에서 문서의 본체(BODY) 를 말하며 문서의 내용이 들어가는 주된 부분이다.
거의 모든 명령 엘리먼트(= 요소) 들이 이곳에 들어간다.
