FORM 태그

- HTML 문서에서 데이터를 주고 받을 때 FORM을 사용하며,
요소들은 < FORM > < /FORM > 사이에 위치해야한다.
- 이름 (name)과 값 (value)이 짝을 지어서 이동한다.
- 여러개의 FORM이 나올 수 있지만 name은 반드시 각각 달라야한다.

요소 이름(name) 값(value)을 임의로 넣음

name='java' gender='여자' marry='미혼' select="selected"

가장 중요한 태그인 INPUT 태그

INPUT 안에서 사용 가능한 속성

- text, password, cheakbox, radio, button,hidden 등
