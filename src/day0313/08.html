특수문자

- 특수문자는 일종의 기호 문자이며 두가지 기능을 위해 사용된다.

1. <HEAD> 표시하면 < HEAD > 로 표현된다
왼쪽의 꺽쇠(< ) 는 태그의 시작, 오른쪽 꺽쇠( >) 는 태그의 종료를 의미한다
앤드기호(&) 는 특수문자의 시작을 알린다.
HTML 문서에서 특수한 의미를 지니고 있으므로,
일반 문서 내부에서는 사용 할 수 없다.

2. 일반적으로 문서 편집기에 없는 문자를 표현하기위해 사용된다.

이중따옴표(")는 "로 사용해도 되며, 고급의 복잡한 문서에서는 반드시 " 로 사용해야 할 경우도 있다.
문자들을 HTML에서 사용하려면, 그 문자들의 문자기호(escape 기능)을 사용해야 한다.
< 혹은 < : (< )의 문자 기호
> 혹은 > : (>)의 문자 기호
  혹은   : ( ) 공백의 문자기호
© 혹은 © : 저작권의 문자기호

∫ 혹은 ∫ : 인테그랄
∑ 혹은 ∑ : 합계
