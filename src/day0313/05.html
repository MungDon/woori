FONT 태그

글자의 표현 형태(스타일) 를 나타내며 여러가지의 특성 및 특성 값(= 속성)을 가질 수 있는데
어트리뷰트(Attribute = 속성)라고한다.
HTML 은 대소문자를 구별하지 않으나,
명령 요소 : ELEMENT (: 상세 내역 목록) 은 대문자로,
속성 : Attribute는 소문자로 표현하여 구분하기 쉽도록하자.

Font의 기본적인 속성

- size, color, face 등이 있다.

폰트색상
색상값을 색명이나 색상의 번호가 된다.
색상 번호는 빨강(red), 초록(green), 파랑(blue) 을 나타내는 rrggbb 색상의 정도를
16진수로 표시한 것으로 높은값이 ff, 낮은값이 00이고,
색상명은 이 번호들을 기억하는 대신 기억하기 좋은 영문으로 미리 정한것이다
rgb 색상표 검색하여 사용가능

Font color = "ff0000"
Font color = "ccccc"
Font color = "gold"
Font color = "blue"

FONT face="궁서" 글자
FONT face="바탕" 글자

내용내용
이런식으로 속성들을 조합하여 사용가능
FONT size="4" color="008000" face="궁서"
FONT size="3" color="000080" face="Comic Sans MS"
FONT size="-1" color="fusia" face="바탕"

속성값을 따옴표("")로 묶어주는것이 원칙! 생략가능
