* 변수 예약어 var 예약어를 쓰지않으면 의도치않게 전역 변수가 될 수 있다 변수를 재선언 하거나 재할당 하는 경우도 발생한다. * var - let/const : 가장큰 차이는 scope(변수가 가지는 범위)이다. var - 함수 영역 전체 범위 let, const - 블럭{} 영역의 범위 *전연 변수 - 변수 이름만 쓰고 값을 할당한다 1. let - 블럭{} 에서만 유효하고 벗어나면 사용하지 못한다 - 재할당은 가능 / 재선언 불가능. 그러니, 예약어 var 와 같이 실수로 같은 변수의 이름을 사용할 걱정이 없다. -> 같은 변수 이름 중복 불가능 2. const - 블럭{}에서만 유효하고 벗어나면 사용 할 수 없다. - 상수변수(constant variable) - 상수 : 프로그램안에서 변하지 않는 값 뜻함 즉, 변하지 않는 값을 변수로 선언할 때 사용함. 재선언 / 재할당이 모두 불가능하다.
