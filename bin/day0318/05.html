* 자료형 - 컴퓨터가 처리할 수 있는 자료의 형태 - 기본 유형 숫자(number) : 정수 / 실수 - 소수점의 유무 : 자바와는 다르게 정수와 실수가 명확하게 구분되지 않는다. 실수 계산이 정확하지 않다. 문자(string) : 따옴표(" ", ' ') 논리형(boolean) : true/ false 소문자로 사용 - 복합 유형 배열(array) : 하나의 변수에 여러개의 값을 저장한다. : 배열명[값1, 값2, 값3] : [] -> 빈 배열 선언 : 방 번호가 index이며 0부터 시작한다. 객체 : 함수와 속성을 함께 포함 한다. - 특수 유형 nudefined : 변수에 할당된 값이 없을 때 : 즉, 선언만 한 상태 null : 변수에 할당 된 값이 유효 하지 않을때의 상태
